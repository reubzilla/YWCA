# Club Leader Guide

English is the canonical version of this guide.

## Access and responsibilities

Club Leaders can use Home and Availability. They can also view the same-day management Dashboard and manage Visitor Schedules. These permissions are checked again by the server for every protected request.

Club Leaders cannot manage Sessions & Events, Members, roles, or Settings.

## Student functions

Use Home and Availability in the same way described in the Student Guide. The navigation opens the Visitor Schedules management view instead of the Student-only personal Volunteer Schedule. Club Leaders are eligible to be assigned as volunteers, and their assignments appear in the management list.

## Use the Dashboard

1. Open **Dashboard**.
2. If more than one active session is taking place today, select the required session.
3. Review the totals and member groups for availability and volunteers.
4. Check the warning list for volunteers who marked themselves `Unavailable`.
5. Review the read-only attendance summary.

The Dashboard contains club-wide availability, private availability notes, volunteer information, and attendance information. Treat this information as confidential and do not share it with Students who are not authorised to see it.

The Dashboard only loads active, non-cancelled sessions taking place today.

## Manage Visitor Schedules

“Visitor Schedule” is the interface name for Volunteer Assignments.

1. Open **Visitor Schedules**.
2. Select an active upcoming session.
3. Review existing assignments and each eligible member's current availability response.
4. Add one or more active Students or Club Leaders.
5. Enter activity, location, departure time, assignment status, and notes as needed.
6. Save the assignment.

The portal prevents duplicate assignments for the same Session ID and Member ID. Assigning someone marked `Unavailable` requires an explicit confirmation. `Unsure` or no response produces a notice but does not block the assignment.

Departure time must be valid and cannot be later than the session end time. It may be earlier than the session start time to allow for travel.

## Edit, cancel, or remove an assignment

- Edit a future assignment to change its managed fields.
- Use `Cancelled` when an assignment should remain in the historical record.
- Permanent deletion is restricted to safe future assignments.
- An assignment for today, a historical assignment, an assignment with attendance history, or an assignment whose session is missing cannot be permanently deleted.
- Visitor Schedule actions never delete associated attendance history.

## Change language

Use the header language selector. Interface labels, dates, warnings, and statuses rerender in Japanese or English. Names, session titles, activities, locations, and notes are not translated.

## Troubleshooting

- If Dashboard shows no session, confirm that an active, non-cancelled session exists for today.
- If a member is unavailable for selection, confirm with a Teacher that the member is active and has the role `Student` or `Club Leader`.
- If deletion is blocked, cancel the assignment instead and preserve its history.
- Ask a Teacher to correct session or member records that a Club Leader cannot administer.
