# 生徒向けガイド

このガイドは、英語版を正本として同じ構成で作成しています。

## アクセスとプライバシー

ポータルは、ログイン中の学校Google Workspaceアカウントと一致する在籍中のメンバー情報から本人を識別します。他の生徒の情報を入力して本人確認を行う仕組みではありません。

生徒が受け取るのは、自分の参加可否と訪問担当の情報だけです。他の生徒の個人メモ、出席情報、個人情報は生徒向け画面に表示されません。

## ホーム

ホームには次の情報が表示されます。

- 自分の氏名、ロール、登録されている場合は学年、学校メールアドレス
- 参加可否が未回答の今後の活動に関する通知
- 自分の今後の訪問担当に関する通知
- 有効な今後の活動、その日付と時刻、活動種別、設定されている場合は回答期限、メモ

通知の日付と画面ラベルは選択した言語に従います。活動名、活動内容、場所、メモは入力されたまま表示されます。

## 参加可否の回答

1. **参加可否**を開きます。
2. 今後の活動を探します。
3. 選択中の言語で表示される、次の内部回答値のいずれかを選びます。
   - `Available`
   - `Unavailable`
   - `Unsure`
4. 必要に応じて、500文字以内の個人メモを入力します。
5. **回答を保存**を選びます。

活動が有効で、中止されておらず、本日以降の日付である間は、後から回答を更新できます。最初の提出日時は保持され、最終更新日時が変更されます。

個人メモは、権限のあるクラブリーダーと教員が確認できます。他の生徒には表示されません。

**Coming Soon（近日公開）：** 表示されている回答期限の自動適用。現在の保存処理は、期限を過ぎたことだけを理由に回答を拒否しません。

## 訪問担当の確認

**訪問スケジュール**を開くと、自分の担当だけが表示されます。各担当には次の情報が表示される場合があります。

- 活動の日付と名称
- 活動内容
- 場所
- 出発時刻
- 担当状態
- 自分向けのメモ

担当状態は内部的に`Assigned`、`Confirmed`、`Declined`、`Cancelled`として保存され、表示時だけ翻訳されます。

## 言語の変更

ページ上部の言語セレクターを使用します。初期設定は日本語で、英語も選べます。現在の画面はすぐに再描画され、選択内容はブラウザに保存されます。

## Coming Soon（近日公開）

- 生徒のチェックイン。
- QRコードと当日の出席手順。

現在のチェックイン画面はプレースホルダーであり、出席は記録されません。

## トラブルシューティング

- ポータルを読み込めない場合は、登録済みの学校アカウントを使用しているか確認してください。
- 氏名、学年、ロールが誤っている場合は、教員にメンバー情報の更新を依頼してください。
- 活動が表示されない場合は、その活動が有効で中止されていないか教員に確認してください。
- 担当内容が誤っている場合は、クラブリーダーまたは教員に連絡してください。
