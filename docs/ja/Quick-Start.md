# クイックスタート

このガイドは、英語版を正本として同じ構成で作成しています。

## 目的

YWCAクラブポータルでは、在籍中のクラブメンバーが活動予定を確認し、参加可否を回答し、自分の訪問担当を確認できます。クラブリーダーと教員には、追加の管理機能があります。

## 利用前の確認

- ポータルの`Members`データに登録されている学校のGoogle Workspaceアカウントを使用してください。
- メンバー情報が在籍中になっている必要があります。
- 生徒がデータベース用スプレッドシートへアクセスする必要はありません。また、生徒にはアクセス権を付与しないでください。

## ログイン

1. 学校から案内されたクラブポータルのデプロイ済みウェブアプリURLを開きます。
2. Googleからログインを求められた場合は、登録済みの学校Google Workspaceアカウントでログインします。
3. ホーム画面に自分の名前、通知、今後の活動が表示されるまで待ちます。

アカウントを識別できない場合は、教員にメールアドレスと在籍状態をメンバー管理で確認してもらってください。

## 言語の選択

初期設定は日本語です。ページ上部の言語セレクターで、日本語と英語を切り替えられます。選択内容はブラウザのローカルストレージに保存されます。

活動名、氏名、メモ、活動内容、場所は入力されたまま表示され、言語を変更しても翻訳されません。

## ロール別ナビゲーション

### 生徒

- ホーム
- 参加可否
- 訪問スケジュール
- チェックイン — **Coming Soon（近日公開）**

### クラブリーダー

クラブリーダーはホームと参加可否に加えて、次の機能を利用できます。

- 訪問予定管理
- ダッシュボード
- 出席管理 — **Coming Soon（近日公開）**

### 教員

教員はクラブリーダー向け機能に加えて、次の機能を利用できます。

- 活動・イベント管理
- メンバー管理
- 設定管理 — **Coming Soon（近日公開）**

## 最初に行うこと

1. ホームで未回答の参加可否や訪問担当の通知を確認します。
2. 参加可否を開き、対象となる活動ごとに`Available`、`Unavailable`、`Unsure`のいずれかを保存します。
3. 訪問スケジュール、または管理者の場合は訪問予定管理を開き、担当を確認します。
4. クラブリーダーと教員は、本日実施される有効な活動についてダッシュボードを開けます。

## Coming Soon（近日公開）

- 手動の出席記録と出席管理。
- 教員向け設定管理。

## 問い合わせ

アカウントが見つからない、在籍中になっていない、ロールが誤っている、または必要な管理画面を開けない場合は、教員に連絡してください。
